- **automazeio/vibeproxy** (24 today, Swift) — "Native macOS menu bar app to use your Claude Code & ChatGPT subscriptions with AI coding tools." - **777genius/claude-notifications-go** (17 today, Go) — "Smart notifications plugin for Claude Code with cross-platform support and multiple notification types." Why relevant: Cross-platform notification layer built specifically for Claude Code events. Shikki currently has no standardized notification surface — this plugin solves the exact problem of surfacing Claude Code state changes (completion, errors, approvals needed) without requiring terminal focus. The Go cross-platform approach is an alternative to Swift-only solutions like claude-island.

- **rivet-dev/rivet** (12 today, Rust) — "Rivet Actors are the primitive for stateful workloads. Built for AI agents, collaborative apps, and durable execution." Why relevant: Proposes a stateful actor primitive as the foundation for AI agent persistence — agent state survives process restarts, retries, and crashes. If Shikki builds longer-running autonomous agents or distributed dispatch pipelines, Rivet Actors are an architectural primitive worth understanding before rolling a custom solution.

- **asgeirtj/system_prompts_leaks** (306 today, 36,535 total) — Extracted system prompts from ChatGPT, Claude, Gemini, Grok, Perplexity, and others. Updated regularly. Why relevant: The most comprehensive public corpus of production AI system prompts in existence. Directly applicable to calibrating Shikki's persona system prompts (@Sensei, @Hanami, etc.), understanding how Anthropic structures Claude's own instructions, and studying prompt engineering patterns at scale from competing systems.
