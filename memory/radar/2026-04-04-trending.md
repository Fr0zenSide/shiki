- **pydantic/monty** (6,657 total, 18 today, Rust) — "A minimal, secure Python interpreter written in Rust for use by AI." Why relevant: Pydantic building a sandboxed Python interpreter for AI agents — same problem space as nono but at the language level rather than kernel level. The complementary approach: nono handles syscall isolation, monty handles Python semantic isolation. Together they represent a full agent execution safety stack. Relevant to any Shikki feature that executes user-provided code.
